Grunnleggende JavaScript (U13)


Oppgave:


Sett opp en funksjon som skal endre hvilken 'class' som er aktiv og sett opp et 'event' som trigges av at noe blir trykket på som gir oss en theme switch med å bytte hvilken CSS som skal synes på nettsiden. Jeg vil spesifikt at dere skal modifisere hvilken 'class'-navn body-elementet har, så kan dere vurdere hvilke element dere vil bruke for å oppnå dette.


Krav:


Hente elementer fra HTML med getElementById(), Modifisere elementene i JS til å gå fra å ha et 'class'-navn til et annet 'class'-navn med .classList.toggle() , Sende de tilbake til HTML med .addEventListener()


Bonus:


Sett opp flere CSS properties som blir manipulert i samme class Eksperimenter med å hente ulike elementer fra HTML og manipuler dem med JavaScript
